## 2. Digital Hygiene Rules

"Digital Hygiene" refers to the practices that keep your files organized, secure, and professional.

### Rule 1: File Naming Conventions
Never send a file named `final_report_v2(1).xlsx`.
Use a standardized format: `[Date]_[ProjectName]_[AuthorInitials]_[Version].xlsx`
*Example: 20261015_Q3Audit_JD_v1.xlsx*

### Rule 2: The "Zero Hardcoding" Rule in Excel
If you are building a financial model, **never** type a raw number into a formula (e.g., `=A1 * 0.18`).
Instead, put `18%` in a separate cell (e.g., `B1`) and reference it: `=A1 * B1`. This allows anyone to update the tax rate without breaking the entire sheet.

### Rule 3: Version Control
If multiple people are editing a document, do not send copies back and forth via email. Use cloud links (Google Drive, OneDrive) or track changes to maintain a "Single Source of Truth."

## 3. Communication Tools

Beyond documents, you must master the tools used to communicate within the company.

*   **Slack / Microsoft Teams:** Used for quick, informal communication. **Rule:** Keep messages brief. Use threads to reply to specific topics so the main channel doesn't get cluttered.
*   **Outlook / Gmail:** Used for formal communication and external clients. (Covered extensively in Sem 4).
